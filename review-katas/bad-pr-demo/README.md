# Bad PR Demo

This folder is an isolated review kata. The code and tests are intentionally low quality so they can be used for review practice.

## What is included

- `src/order-utils.js` contains multiple review-worthy problems.
- `test/order-utils.test.js` includes brittle and flaky tests on purpose.

## Run

```bash
cd review-katas/bad-pr-demo
npm test
```

## Important

Do not copy this code into production services. It is only for exercising review workflows, PR comments, and bug-spotting.
