# notes/

Scratch space for workshop participants. Save your design briefs, repro steps,
investigation notes, and prompt captures here. Some lab steps explicitly write
to specific files in this directory:

- `notes/cold-prompt.md` — Lab 1 cold prompt capture
- `notes/design.md` — Lab 1 design brief
- `notes/regression.md` — Lab 1 review regression find / Lab 2 culprit commit
- `notes/repro.md` — Lab 2 reproduction steps
- `notes/investigation.md` — Lab 2 sub-agent findings
- `notes/fix.md` — Lab 2 fix rationale
- `notes/prompts.md` — prompt captures (reference branches use this)

This directory is checked in. If you write something you don't want shared,
delete it before pushing.
